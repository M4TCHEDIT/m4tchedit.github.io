# Bebidas Express - Tienda de Bebidas

Distribuidora de bebidas al por mayor y menor.

- Funcionalidades: carrito, promociones, panel de admin.
- Tecnologías: HTML5, CSS3, JavaScript (vanilla).
- Responsive y listo para producción.

## Cómo usar
1. Clona este repositorio.
2. Abre `index.html` en un navegador (recomendado: Chrome).
3. O súbelo a GitHub Pages para acceso público.
